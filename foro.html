¿Cuál es el mejor lenguaje para comenzar en programación?

Por AnaTech • Hace 3 horas • Programación y Desarrollo

Hola a todos, estoy interesado en comenzar a aprender programación pero no sé por dónde empezar. He investigado un poco y veo que hay muchos lenguajes: Python, JavaScript, Java, C++, etc.

Me gustaría saber cuál recomiendan para un principiante absoluto. ¿Qué lenguaje es más amigable para empezar? ¿Cuál tiene más salida laboral? ¿Es mejor empezar con uno más sencillo aunque tenga menos aplicaciones?

Agradezco cualquier consejo o experiencia que puedan compartir. ¡Gracias!

Respuestas (2)

CM
CodeMaster
Hace 2 horas

Hola, en mi opinión Python es excelente para empezar. Tiene una sintaxis clara y legible que facilita el aprendizaje. Además, es muy versátil y se usa en muchos campos: web, data science, inteligencia artificial, automatización, etc.

Otra ventaja es que tiene una comunidad enorme con muchos recursos para aprender.

DR
DevRosa
Hace 1 hora

Coincido con CodeMaster sobre Python, pero también consideraría JavaScript si te interesa el desarrollo web. Con JavaScript puedes hacer tanto frontend como backend, y ver resultados visuales rápidamente puede ser muy motivador para un principiante.

Al final, lo más importante es empezar con algo y ser constante. ¡Mucha suerte!
